You need to install some modules to start:
  geopy.geocoders      |             |    pip install geopy
  timezonefinder        \   -->     /     pip install timezonefinder
  datetime               \  Win+R  / -->  pip install datetime
  requests               /  cmd    \      pip install requests
  pytz                  /           \     pip install pytz
  PIL                  |             |    pip install pillow
  
  Notice: (!)
    API key may not work. If you want continue using it you should buy "Developer" plan at openweathermap.org
